Side Menu

Guia de configuração do menu lateral da aplicação.

O componente <gems-side-menu> é o pilar de navegação principal em aplicações corporativas.

Configuração (config)

A propriedade [config] recebe um objeto do tipo GemsSideMenuConfig que define o título e os itens do menu:

menuConfig: GemsSideMenuConfig = {
  headerTitle: 'GEMS SDK',
  items: [
    { label: 'Início', icon: 'fa-solid fa-house', route: '/' }
  ]
};

Dados de Usuário

Na parte inferior do menu, você pode fornecer os dados do usuário autenticado:

  • [userName]: O nome exibido (ex: 'Administrador')
  • [userEmail]: O email exibido (ex: 'admin@gems.com')
  • [userInitials]: Sigla para o avatar (ex: 'AD')
  • (logoutClick): Evento disparado no botão de sair

Responsividade (Desktop vs Mobile)

O comportamento mobile é ativado pela classe .mobile-menu-open aplicada em uma div .showcase-layout. No app.component.html, este controle é manual para permitir interação com um possível navbar superior:

<div class="showcase-layout" [class.mobile-menu-open]="mobileMenuOpen">
  <gems-side-menu ...></gems-side-menu>
  <main class="showcase-content">
    <router-outlet></router-outlet>
  </main>
</div>
