HTTP / BaseStore

GemsBaseStore é uma classe abstrata que centraliza chamadas HTTP com autenticação Keycloak opcional e suporte a paginação.

Forneça a URL da API via token de injeção GEMS_API_URL. Todas as stores herdam esse valor.

URL automática

Cada store recebe o path do controller no super(). Os métodos constroem a URL final como GEMS_API_URL + controller + endpoint.

Keycloak opcional

Se KeycloakService estiver presente (optional: true), o token é renovado e adicionado como Authorization: Bearer … automaticamente.

Paginação com GemsPageable

Passe { pageable } nas opções dos métodos GET para adicionar os query params page, size e sort.
