Tema dinâmico

Configure cores, paletas e troque o tema em runtime com provideGemsTheme.

A SDK aplica automaticamente CSS custom properties (--gems-*) no :root a partir de poucas cores base. Não é necessário configurar cada shade manualmente.

Após chamar provideGemsTheme, as seguintes variáveis ficam disponíveis globalmente:

Paleta primária

--gems-primary-50 → --gems-primary-900

Semânticos

--gems-danger-*, --gems-success-*, --gems-warning-*, --gems-info-*

Tipografia

--gems-font-size-xs → --gems-font-size-4xl
--gems-font-weight-*, --gems-line-height-*

Espaçamento

--gems-spacing-xs → --gems-spacing-2xl

Bordas & sombras

--gems-radius-*, --gems-shadow-*
--gems-border-color

Z-index

--gems-z-modal (500), --gems-z-toast (800)
--gems-z-modal-backdrop (400)
