Instalação da SDK

Aprenda como adicionar e configurar a GEMS SDK no seu projeto Angular.

1. Registry do GitHub Packages

Crie ou edite o arquivo .npmrc na raiz do projeto:

@gabriel-mdias:registry=https://npm.pkg.github.com

2. Instalação via NPM

Instale utilizando a flag --legacy-peer-deps devido à resolução estrita do Angular 20:

npm install @gabriel-mdias/angular-gems-sdk --legacy-peer-deps

3. Estilos e Theming

Adicione os estilos globais em styles.css:

@import "@gabriel-mdias/angular-gems-sdk/core/tokens/gems-design-tokens.css";
@import "@gabriel-mdias/angular-gems-sdk/core/tokens/gems-utilities.css";
@import "@gabriel-mdias/angular-gems-sdk/core/tokens/gems-animations.css";

Injete o provedor de tema dinâmico em app.config.ts:

import { provideGemsTheme } from '@gabriel-mdias/angular-gems-sdk/core/theme';

export const appConfig: ApplicationConfig = {
  providers: [
    provideGemsTheme({
      primary: '#0B5FFF',
      secondary: '#4A5568',
      tertiary: '#E2E8F0',
      background: '#F7FAFC'
    })
  ]
};
